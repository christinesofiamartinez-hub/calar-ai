# Calar

AI-powered business intelligence for SMBs.

## Deploy to Vercel

1. Push this repo to GitHub
2. Go to vercel.com → New Project → Import repo
3. Vercel auto-detects Next.js → click Deploy
4. Add calar.me in Settings → Domains

## Local development

```bash
npm install
npm run dev
```
